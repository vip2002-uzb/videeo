# Universal Video Yuklovchi Bot

Bu Telegram bot quyidagi platformalardan videolarni yuklab beradi:
- Instagram
- Facebook
- YouTube
- TikTok
- Threads

## O'rnatish

1. Talablar: Python 3.8+ o'rnatilgan bo'lishi kerak.
2. Kutubxonalarni o'rnatish:
   ```bash
   pip install -r requirements.txt
   ```

## Sozlash

1. Telegramda @BotFather ga kiring va yangi bot yarating.
2. Berilgan TOKEN ni `main.py` faylidagi `BOT_TOKEN` o'zgaruvchisiga yozing:
   ```python
   BOT_TOKEN = "123456789:ABCdefGHIjklMNOpqrstUVwxyz"
   ```

## Ishga tushirish

Botni ishga tushirish uchun quyidagi buyruqni bering:
```bash
python main.py
```
Bot ishga tushgach, unga video havolasini yuboring.
